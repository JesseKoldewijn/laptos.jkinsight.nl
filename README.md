> A blazingly fast Leptos website

## Install

```sh
rustup toolchain install nightly
rustup default nightly
rustup target add wasm32-unknown-unknown
cargo install trunk --locked
```

## Usage

### Development server

```sh
trunk serve --open
```

### TailwindCSS

```sh 
pnpm run compile-css
```
